## Overview

This document outlines the task assignments for v1.1 development, following the correct architecture where **opencode is the master agent** and **knowledge-assistant provides specialized tools**.

---

## Architecture Summary

```
opencode (Master)
  ├── File operations (own capability)
  ├── NLU & intent understanding (own capability)
  └── Calls knowledge-assistant tools
      ↓
knowledge-assistant (Tool Library)
  ├── build_semantic_index(documents) → IndexResult
  ├── semantic_search(query) → [SearchResult]
  ├── extract_keywords(content) → [Keyword]
  └── EmailConnector → Email data
```

## Team Assignments

### Data Team

**Sprint 1 Tasks**:

#### TASK-D1: Semantic Index Builder
**Priority**: P0  
**Estimated Effort**: 5 days  
**Dependencies**: None  

**Description**: Implement `build_semantic_index()` function

**Requirements**:
- Input: List of document data (path, content, metadata) from opencode
- Process: Generate embeddings, build FAISS index
- Output: IndexResult with statistics
- NO file scanning or reading (opencode provides data)

**Acceptance Criteria**:
- [ ] Function accepts document list
- [ ] Generates vector embeddings
- [ ] Builds FAISS index
- [ ] Saves index to disk
- [ ] Returns structured result
- [ ] Unit tests > 85% coverage
- [ ] Performance: 1000 docs < 30s

**Files to Create/Modify**:
```
scripts/
├── embeddings/
│   ├── __init__.py
│   ├── encoder.py              # NEW: Embedding encoder
│   └── models.py               # NEW: Model management
├── index/
│   ├── __init__.py
│   ├── vector_store.py         # NEW: FAISS vector store
│   └── manager.py              # NEW: Index manager
└── tools/
    ├── __init__.py
    └── indexing.py             # NEW: build_semantic_index()
```

**API Example**:
```python
from scripts.tools.indexing import build_semantic_index

# opencode provides document data
documents = [
    {
        "path": "D:/notes/python.md",
        "content": "...",
        "metadata": {"title": "...", "date": "..."}
    }
]

result = build_semantic_index(
    documents=documents,
    index_path=".ka-index"
)

# Returns
{
    "success": True,
    "total_docs": 156,
    "total_chunks": 489,
    "index_size": "23.5 MB",
    "build_time": "12.3s"
}
```
