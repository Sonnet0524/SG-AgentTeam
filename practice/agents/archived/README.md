## Why Archived?

The team structure was reorganized for v1.1 to:

1. **Clearer Responsibilities**
   - Core Team: Core data processing tools
   - AI Team: Vector embeddings and semantic search (NEW)
   - Integration Team: opencode integration and connectors

2. **Better Skill Matching**
   - AI Team requires ML/NLP skills
   - Integration Team requires opencode knowledge
   - Core Team focuses on traditional data processing

3. **Support v1.1 Features**
   - Semantic search requires dedicated AI expertise
   - opencode integration needs dedicated focus
   - Clearer separation of concerns
