# 与其他方法对比

> 🔍 Agent Team Framework与其他Agent协作方法的对比

---

## 对比对象

### Multi-Agent System (MAS)

**相似点**:
- 多Agent协作
- 任务分解和分配

**不同点**:
- 我们使用文档作为交互媒介
- 我们强调边界隔离
- 我们基于Token管理进度

---

### AutoGPT / BabyAGI

**相似点**:
- AI Agent自主执行任务
- 任务规划和分解

**不同点**:
- 我们有明确的Agent角色分工
- 我们使用文档化交互
- 我们有清晰的边界隔离

---

### LangChain / AutoGen

**相似点**:
- Agent协作框架
- 工具使用能力

**不同点**:
- 我们聚焦文档化交互
- 我们有Token-Based管理
- 我们的研究导向更强

---

## 核心差异

**Agent Team Framework的核心创新**:

1. **文档化交互**: 基于文档的上下文交换
2. **Message Queue愿景**: 为全自动交互奠基
3. **Token-Based管理**: 精确的工作量度量
4. **边界隔离**: 零冲突的协作模式
