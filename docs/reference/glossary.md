# 术语表

> 📚 Agent Team Framework核心术语定义

---

## A

### Agent
AI Agent，能够自主执行任务的智能体。

### Agent First
信息表达原则，优先考虑机器可读性和效率。

---

## C

### Context
Agent启动时加载的信息集合。

### Context最小化
通过分层披露最小化context占用的方法论。

---

## D

### Document Hierarchy
文档分层体系，将文档分为Level 0/1/2三级。

---

## H

### Human First
信息表达原则，优先考虑人类可读性和友好性。

---

## M

### Message Queue
消息队列，未来实现Agent自动交互的机制。

---

## T

### Token
AI的计算单位，可作为工作量度量。

### Token-Based Management
基于Token的工作量管理和进度跟踪方法。
